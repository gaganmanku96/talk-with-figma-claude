Claude Talk to Figma - Setup Guide

First time setup: Follow these steps to connect Claude Desktop App with Figma.

Step 1: Setup Figma Plugin

  1. In Figma, go to Plugins > Development > New Plugin
  2. Choose "Link existing plugin"
  3. Select the src/cursor_mcp_plugin/manifest.json file from this installation
  4. The plugin should now be available in your Figma development plugins

Step 2: Start the Servers

The servers are now running in the background. Please keep the terminal window open!

Step 3: Configure Claude Desktop App

  1. Open Claude Desktop App
  2. Go to Settings > MCPs
  3. Add a new MCP called "ClaudeTalkToFigma" with these settings: 
    Command: node
Args: /home/gagan/projects/talk_to_figma_claude/mcp-server.js
Working directory: /home/gagan/projects/talk_to_figma_claude
  4. Enable the MCP by toggling it on

Step 4: Using the Integration

  1. Open Figma and run the plugin (from the Plugins > Development menu)
  2. In the plugin, click "Generate Channel" and copy the channel ID
  3. In Claude, send the command: join_channel {"channelId": "PASTE_CHANNEL_ID_HERE"}
  4. You're now ready to control Figma with Claude!

Having issues? Check the logs in the terminal window for detailed error messages.

Example Commands

After joining a channel, try these commands in Claude:

get_document_info

get_selection

create_rectangle {"x": 100, "y": 100, "width": 200, "height": 100, "name": "My Rectangle"}

create_text {"x": 100, "y": 300, "text": "Hello from Claude!", "fontSize": 24}
